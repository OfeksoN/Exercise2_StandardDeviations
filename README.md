Overview
This project focuses on fundamental statistical concepts and operations, including:

Mean (Average): Calculating the central tendency of a dataset.
Standard Deviation: Measuring the spread or variability of data.
Sample Space: Representing all possible outcomes of an experiment.
File Handling: Reading data from CSV and other file formats for analysis.

The goal is to provide a simple yet powerful toolkit for performing basic statistical computations and probability-related tasks in Python.
